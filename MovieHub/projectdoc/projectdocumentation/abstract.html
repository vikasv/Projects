Abstract

About the project

Live without celebration is a long road without Inns” said a famous philosopher. People celebrate various special occasion and moments with their beloved ones in different styles. Among these various styles, going to a movie has become a common trend lately. Being a film buff myself, I find watching movies amusing and watching it in a movie theatre adds to the experience. Movies are definitely a huge source for entertainment, stress relief, awareness, good laugh, thrilling experience, etc… So, I wanted to build a website related to movies that helps people keep abreast of the latest updated related to movies. My project provides information related to latest and upcoming movies, release dates, movies currently in theatres, movie posters, and also a search functionality to find the information with respect to a particular movie.

Technologies used

jQuery Client Side
Ajax Client Side
HTML Client side
CSS3 Styling
Java Script client Side

APIs called

Rotten Tomatoes API To get the various list of movies which include information top movies running in theatres, Weekend releases and future releases
MyApiFilms API Used to retrieve information about a movie requested in the search by the user
Youtube API To retrieve the movie trailers based on the Genre.

Functionalities implemented

  • Display movie currently in theatres which includes a lot of information about each movie, say, date of release, synopsis of the movie, cast, length, etc.
  • Showcase highest grossing movies at the Box Office so far in the descending order.
  • Clicking on the movie poster takes you to IMDB page if one would like to see the ratings and reviews of the movie.
  • Movies releasing this weekend
  • Showing list of trailers from youtube based on the genre
  • Future releases
